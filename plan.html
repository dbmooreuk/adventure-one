Item Types

item Pickable items that go into inventory. Can be used with other items or targets.
target Objects in the scene that items can be used on. Cannot be picked up.
link Navigation links that appear after certain actions. Click to travel to another scene.
decor Decorative objects that can be examined but not picked up or used. Adds atmosphere to scenes.

Outcomes

keep Item stays in inventory after use. Can be reused multiple times.
remove Item is consumed/removed from inventory after use. One-time use only.
removeTarget Target is removed from the scene after use (only works on targets).
scene Using this item/link triggers scene navigation or unlocks a new scene.
Multiple: Can use array like ["scene", "remove", "removeTarget"] to combine outcomes.
Item + Target: When using item on target, "remove" is checked on BOTH. Other outcomes use target's value.

Use Result

Purpose: The name of the new item/link created when two items are combined or used together.
Example: Using "oar" on "boat" creates "crossing" (a link item that appears in the scene).

Scene Properties

Locked: Scene cannot be accessed until unlocked by a specific action/item.
Unlocked By: The useResult name that unlocks this scene (e.g., "crossing" unlocks scene3).
Link To Scene: For link-type items, the scene ID they navigate to when clicked.

Animation System

Transform: CSS transform animations: bob (vertical), pulse (scale), spin (rotate), fade (opacity).
Random: Creates multiple clones of the item that move randomly around the screen with bouncing physics.
Sprite Frames: Array of image filenames that cycle through for frame-by-frame animation.
Sprite Sheet: Single image with multiple frames arranged horizontally, animated via background-position.
FPS: Frames per second for sprite animations (default: 12).
Amplitude: Intensity of transform animations in pixels or percentage (default: 10).
Speed: Speed multiplier for animations (default: 1).
Rotation: Rotation speed for random animations, 0-9 scale (0=no rotation, 5=default, 9=fast).

Achievement System

üèÜ Achievement Items/scenes with this badge add journal entries when triggered.
Scene Achievement: Regular scenes: triggers 4 seconds after entering (first visit only). Puzzle scenes: triggers on completion.
Item Achievement: Triggers when successfully combining this item with another item.
Journal: Players can view all achievements in the Journal modal. Journal button pulses when new achievement is added.

Click Interactions

onClickEffect: Visual effect when clicked: flash (brightness), bounce (jump), shake (horizontal).
onClickSound: Sound effect to play: useItem, pickup, success, error, etc.
triggerPuzzle: Object with {module, config} to launch a puzzle when clicked.
hitW / hitH: Larger touch hitbox dimensions for mobile (visual size stays same).
