Item Types

item Pickable items that go into inventory. Can be used with other items or targets.
target Objects in the scene that items can be used on. Cannot be picked up.
link Navigation links that appear after certain actions. Click to travel to another scene.

Outcomes

keep Item stays in inventory after use. Can be reused multiple times.
remove Item is consumed/removed from inventory after use. One-time use only.
removeTarget Target is removed from the scene after use (only works on targets).
scene Using this item/link triggers scene navigation or unlocks a new scene.
Multiple: Can use array like ["scene", "remove", "removeTarget"] to combine outcomes.
Item + Target: When using item on target, "remove" is checked on BOTH. Other outcomes use target's value.

Use Result

Purpose: The name of the new item/link created when two items are combined or used together.
Example: Using "oar" on "boat" creates "crossing" (a link item that appears in the scene).

Scene Properties

Locked: Scene cannot be accessed until unlocked by a specific action/item.
Unlocked By: The useResult name that unlocks this scene (e.g., "crossing" unlocks scene3).
Link To Scene: For link-type items, the scene ID they navigate to when clicked.
